## 📘 DSA Practice – Array Problems

### 🧠 Overview

In this session, I practiced and understood three fundamental array-based problems:

* Two Sum
* Contains Duplicates
* Best Time to Buy and Sell Stock

These problems helped strengthen my understanding of **hashing techniques and greedy algorithms**, which are commonly used in coding interviews.

---

## 🔢 Problems Covered

### 1. Two Sum

* Learned how to efficiently find pairs using **hashing**
* Improved from brute force approach to optimized solution
* Focus on reducing time complexity using extra space

---

### 2. Contains Duplicates

* Understood how to detect duplicates using a **set**
* Learned importance of fast lookup operations
* Practiced handling simple edge cases

---

### 3. Best Time to Buy and Sell Stock

* Learned **greedy approach** for maximizing profit
* Tracked minimum value dynamically
* Understood real-time decision making in a single pass

---

## ⚡ Key Concepts Learned

* HashMap / Dictionary for quick lookups
* Set for duplicate detection
* Greedy algorithms for optimization
* Importance of time and space complexity
* Transition from brute force → optimized solutions

---

## 🚀 Takeaways

* Efficient problem solving requires choosing the right data structure
* Most array problems can be optimized to **O(n)**
* Practicing patterns is more important than memorizing solutions
